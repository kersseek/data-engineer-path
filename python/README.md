# Python

Este directorio está dedicado a ejercicios de práctica del lenguaje de programación.

## Secciones

### [Empleados](poo/employees.py)

Esta sección contiene clases para gestionar: empleados "Empleado", desarrolladores "Desarrollador" y gerentes "Gerente". Permite crear empleados, modificar su salario, mostrar información y manejar la relación de un gerente con sus empleados.

![POO](https://img.shields.io/badge/POO-FF6F61?style=flat&logoColor=white)

### [Jugador](poo/player.py)

Esta sección contiene clases para personajes de un juego: personajes "character", guerreros "warrior" y magos "wizard". Permite crear personajes, modificar atributos como la vida y calcular el daño de ataque de manera específica para cada tipo de personaje.

![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)
![POO](https://img.shields.io/badge/POO-FF6F61?style=flat&logoColor=white)

### [Escuela](poo/school.py)

Esta sección contiene clases para gestionar personas en un entorno académico: personas "Persona", alumnos "Alumno", profesores "Profesor" y cursos "Curso". Permite crear alumnos y profesores, inscribir alumnos en cursos, asignar cursos a profesores y mostrar información de estudiantes, docentes y cursos.

![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)
![POO](https://img.shields.io/badge/POO-FF6F61?style=flat&logoColor=white)

### [Vehículos](poo/vehicles.py)

Esta sección contiene clases para gestionar vehículos: vehículos "Vehiculo", autos "Auto" y motos "Moto". Permite crear vehículos, modificar su precio, mostrar información y calcular impuestos según el tipo de vehículo y sus características.
